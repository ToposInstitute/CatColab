#### User inputs

<dl>
  <dt>Initial value: $\mathbb{R}_{\geqslant0}$</dt>
  <dd>The initial value of the concentration</dd>
  <dt>Mass conservation: `True | False`</dt>
  <dd>Whether or not flows should preserve mass</dd>
  <dt>Rate: $\mathbb{R}_{\geqslant0}$</dt>
  <dd>*(Only if **Mass conservation** = `True`)* The rate coefficient ($r$) of the reaction</dd>
  <dt>Rate granularity: `Per transition | Per place`</dt>
  <dd>*(Only if **Mass conservation** = `False`)* If flows can have multiple inputs/outputs (e.g. in the case of Petri nets) then rates can be given per flow or per individual input/output</dd>
  <dt>Consumption: $\mathbb{R}_{\geqslant0}$</dt>
  <dd>The consumption rate coefficient ($\kappa$), either per transition (flow) or per place (input/output) depending on **Mass conservation**</dd>
  <dt>Production: $\mathbb{R}_{\geqslant0}$</dt>
  <dd>The production rate coefficient ($\rho$), either per transition (flow) or per place (input/output) depending on **Mass conservation**</dd>
  <dt>Duration: $\mathbb{R}_{\geqslant0}$</dt>
  <dd>The total duration of the simulation in units of time</dd>
</dl>
