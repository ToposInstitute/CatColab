#### User inputs

<dl>
  <dt>Layout: `Directed | Undirected`</dt>
  <dd>Switches between Graphviz layout engines: `Directed` enables [`dot`](https://graphviz.org/docs/layouts/dot/) and `Undirected` enables [`neato`](https://graphviz.org/docs/layouts/neato/)</dd>
  <dt>Direction: `Vertical | Horizontal`</dt>
  <dd>(Only if Layout = `Directed`). Switches between orienting the graph from top to bottom and from left to right.</dd>
</dl>
