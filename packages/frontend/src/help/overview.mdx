# Overview

- CatColab is a **collaborative** environment for **formal**, **interoperable**, **conceptual** modeling.
  You can read about what we mean by these words in the [**Fundamentals of CatColab**](guides/fundamentals) guide.

- CatColab is an open-source software project.
  You can [find us on GitHub ↗](https://github.com/ToposInstitute/CatColab) or [chat with the community on Zulip ↗](https://catcolab.zulipchat.com).

- CatColab is a technology created by Topos Institute and our partners.
  You can [view the credits](credits) or [read more about us ↗](https://topos.institute).

<Warning>
CatColab is under active development with new features added regularly. We aim
to preserve the integrity of your data but you should not store anything
critical or sensitive.
</Warning>


## Using CatColab

The general process of using CatColab looks something like the following:

1. Select a **logic**
2. Build a **model** within the above logic
3. Run various **analyses** on your model.

For more information, see the [guides](guides) or dive into the help pages for [logics](logics).
