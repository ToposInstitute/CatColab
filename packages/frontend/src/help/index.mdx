## What is CatColab?

- a collaborative environment for *formal, interoperable, conceptual modeling*
- an open source software project
  * [find us on
  GitHub](https://github.com/ToposInstitute/CatColab)
  * [chat with the community on Zulip](https://catcolab.zulipchat.com)
- a technology created by the [Topos Institute](https://topos.institute/) and our
  partners
  * <A href="credits">read the credits</A>


## Using CatColab

The general process of using CatColab looks something like the following:

1. Select a **logic**
2. Build a **model** within the above logic
3. Run various **analyses** on your model.
